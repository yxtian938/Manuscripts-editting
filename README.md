# AI Writing Skills

This repository contains reusable Markdown-based AI editing skills.

## Available Skills

### Natural Style Editor

Location: `natural-style-editor/SKILL.md`

Use this skill to revise prose so that it sounds less mechanical, less inflated, and more rhythmically varied while preserving the author's meaning.

## How to use

Copy the content of the relevant `SKILL.md` file and paste it into your AI tool as an editing instruction.

Example:

"Use the natural-style-editor skill below to revise my paragraph. Preserve meaning and academic tone."
